## 通路實驗區（跨實驗複利資產，90 天時鐘）

### C1. 摸魚記（匿名）vs 實名 IG — 雙通路定位實驗
**建立日期**：2026-07-11
**狀態**：🟢 進行中
**設計**：不是同內容 A/B（兩帳號受眾與約束不同），是**分工實驗**——各自用最適定位跑 90 天，看哪條產生可轉化的受眾

| | 摸魚記（匿名） | 實名 IG（主力帳號） |
|---|---|---|
| 定位 | build in public：KIWI 實跑紀錄、產品實驗過程、投資框架思考 | 方法論編譯／AI 工作流／跨域洞察＋link-in-bio 專案集 |
| 合規 | 匿名，可談投資觀察（仍不做個股喊單） | ⚠️ **不碰個股建議**；實名投資內容先過雇主 compliance |
| 優勢假設 | 內容自由度高、題材（AI×投資）稀缺 | 既有人脈網放大、信任轉化率高（BD 資產） |
| 主指標 | 追蹤成長率＋互動率 | link-in-bio 點擊＋私訊詢問數（質 > 量） |

**共同動作**：
- [ ] 實名 IG 建 link-in-bio 專案集頁（一個連結收攏所有 project）——成本近零，先做
- [ ] 摸魚記固定節奏發布（頻率自訂但寫死，例：每週 ≥2 則）
- [ ] 每月底記錄兩邊指標到本卡

**驗證時鐘**：2026-10-11（90 天）
**判定標準**：
- 摸魚記：追蹤數成長 ≥3 倍基期 或 出現自然轉發/被引用 → 加碼
- 實名 IG：link-in-bio 產生 ≥5 個有意義的入站對話（詢問/合作/客戶線索）→ 加碼
- 兩者皆未達 → 檢討內容形式而非放棄通路（通路是複利資產，殺的門檻比產品高）
