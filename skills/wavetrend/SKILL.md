# WaveTrend Oscillator 技術分析 Skill

**觸發指令**：`/wavetrend $TICKER` 或 `/wavetrend analyze $TICKER`

**用途**：用 WaveTrend Oscillator 判斷個股或指數的超買/超賣狀態與進出場時機  
**適用市場**：美股、台股、加密貨幣均適用  
**配合使用**：建議搭配 KIWI ACT index（AVI/CRI/TSI）一起判讀，WaveTrend 看個股時機，ACT 看大盤風險

---

## WaveTrend 是什麼？

WaveTrend Oscillator（WT）是一個基於價格動能的技術指標，由 LazyBear 在 TradingView 發表。  
它把價格的「超買/超賣」狀態轉化為震盪器，類似 RSI 但更敏感於趨勢轉折點，特別擅長抓  
**動能反轉的早期訊號**。

---

## 計算公式

```
# 輸入參數
n1 = 10   # 通道週期（Channel Length）
n2 = 21   # 平均週期（Average Length）

# 第一步：計算平均價格
hlc3 = (high + low + close) / 3

# 第二步：EMA 平滑
esa = EMA(hlc3, n1)

# 第三步：平均偏差
d = EMA( |hlc3 - esa|, n1 )

# 第四步：歸一化成通道指數
ci = (hlc3 - esa) / (0.015 × d)

# 第五步：產出 WaveTrend 主線（WT1）
wt1 = EMA(ci, n2)

# 第六步：訊號線（WT2）
wt2 = SMA(wt1, 4)

# 第七步：差值柱（可選）
diff = wt1 - wt2
```

## 關鍵水位（門檻）

| 水位 | 數值 | 含義 |
|------|------|------|
| 極度超買 | +60 | 強力賣出訊號區域 |
| 超買 | +53 | 開始警惕上漲動能 |
| 中軸 | 0 | 多空分界線 |
| 超賣 | -53 | 開始留意反彈機會 |
| 極度超賣 | -60 | 強力買入訊號區域 |

---

## 訊號邏輯

### 買入訊號（由強到弱）
1. **黃金交叉 + 極度超賣**：WT1 向上穿越 WT2，且兩者都在 -60 以下 → 最強
2. **黃金交叉 + 超賣**：WT1 向上穿越 WT2，且在 -53 以下 → 強
3. **WT1 從負轉正**（穿越 0 軸向上）→ 中等

### 賣出訊號（由強到弱）
1. **死亡交叉 + 極度超買**：WT1 向下穿越 WT2，且兩者都在 +60 以上 → 最強
2. **死亡交叉 + 超買**：WT1 向下穿越 WT2，且在 +53 以上 → 強
3. **WT1 從正轉負**（穿越 0 軸向下）→ 中等

### 背離訊號（領先指標）
- **牛市背離**：價格創新低，但 WT1 沒有創新低 → 反彈預警
- **熊市背離**：價格創新高，但 WT1 沒有創新高 → 回調預警

---

## 分析步驟

### Step 1：確認目前水位
- WT1 和 WT2 目前在哪個區間（超買/超賣/中性）？
- 是否接近關鍵水位（±53 或 ±60）？

### Step 2：找交叉訊號
- WT1 最近有沒有穿越 WT2？
- 穿越時的水位（是否在超買/超賣區）？
- 交叉方向（向上=潛在買入，向下=潛在賣出）

### Step 3：確認背離
- 最近的高點/低點，WT1 有沒有同步？
- 有無牛市/熊市背離？

### Step 4：結合大盤風險（KIWI ACT）
| ACT 狀態 | WaveTrend 超賣買入訊號 | WaveTrend 超買賣出訊號 |
|----------|----------------------|----------------------|
| CRI 低、TSI 低（低風險） | 可以積極執行 | 參考用，不急賣 |
| CRI 高 或 TSI 高（高風險） | 謹慎，減少部位 | 積極執行，保護獲利 |
| AVI 高（市場貴） | 只買最確定的訊號 | 更嚴格執行 |

### Step 5：輸出結論
格式：
```
[股票代號] WaveTrend 分析

當前狀態：WT1=[數值] / WT2=[數值] / 區間=[超買/中性/超賣]
訊號：[黃金交叉/死亡交叉/中性/背離]
強度：[強/中/弱]
建議：[買入觀察/持有/賣出觀察/無操作]
搭配 ACT：[目前 CRI/TSI 狀態，對訊號的影響]
注意事項：[任何特殊情況或需要人工確認的點]
```

---

## 限制與注意事項

- WaveTrend 是**落後指標**，確認趨勢用，不是預測用
- 在強趨勢市場中，超買/超賣可以持續很長時間（不要只因為超買就賣）
- 最佳用法：**等交叉訊號出現後才行動**，不要搶在交叉前
- 訊號在日線（1D）最可靠；週線（1W）用於確認大趨勢；小時線（1H）噪音大
- 永遠搭配基本面或其他指標（如 ACT index）一起判斷，不要單一指標決策
