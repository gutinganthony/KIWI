# 台灣 × 印度 Delhi：三大創業機會分析
**給有創業夢的你與你的 Delhi 朋友**
*製作日期：2026 年 5 月*

---

## 背景與前提

| | 你（台灣） | 你的朋友（Delhi） |
|--|-----------|----------------|
| 位置 | 台灣 | 印度德里 |
| 背景 | MBA（荷蘭） | MBA（荷蘭）、家庭資源充足 |
| 角色 | 台灣端：供給、技術、製造資源 | 印度端：市場、落地、政府關係 |

**核心邏輯**：台灣有製造技術與供應鏈，印度有龐大市場與政策資金。你們兩人站在兩端，天然形成「資訊套利」的優勢。

## 機會一：Noida 電子製造供應鏈橋接

### 是什麼？
幫台灣中小型電子零組件廠商（連接器、PCB、散熱模組、電源）進入 Noida 電子製造聚落，對接 Dixon Technologies 等印度大型代工廠的採購需求。

### 為什麼現在是好時機？
- 印度電子製造業在 PLI（生產激勵計畫）補貼下高速擴張
- Dixon Technologies 幫蘋果、三星、Motorola 代工，年營收已達 **$15 億美元（2024）**，預計 2027 年達 $30 億
- 這些代工廠的台灣零組件採購仍透過中間商，利潤空間大
- Noida 距離 Delhi 僅 30 分鐘，你朋友地理優勢明顯

### 市場規模
| 指標 | 數字 |
|------|------|
| 印度電子製造業產值（2024） | **$1,010 億美元** |
| 2030 年目標產值 | **$5,000 億美元** |
| Noida/NCR 佔全印電子產值比例 | 約 35–40% |
| 台灣電子零組件年出口值 | **$650 億美元**（其中進入印度比例 < 3%，空間極大） |

### 商業模式與財務預估
```
收入模式：交易撮合佣金（2–5%）+ 月費顧問服務

保守情境（第 1–2 年）：
  代理 5 家台灣廠商 × 平均年交易額 $50 萬美元
  佣金率 3% = 年收入 $7.5 萬美元（約 NT$240 萬）

成長情境（第 3 年）：
  代理 20 家廠商 × 平均年交易額 $100 萬美元
  佣金率 3% = 年收入 $60 萬美元（約 NT$1,920 萬）

啟動成本：極低（主要是差旅與人脈建立），估計 $1–2 萬美元即可驗證
```

### 你朋友需要做什麼？
1. 去 Noida Sector 63/65 拜訪 Dixon、Amber Enterprises 等廠的採購部門
2. 建立「我能幫你找台灣供應商」的定位
3. 收集採購需求清單回來，你在台灣對應找供應商

### 風險
- 印度採購決策慢，付款條件差（Net 60–90 天）
- 需要一定時間建立信任才能拿到訂單
- 台灣廠商可能繞過你直接接觸

**風險評級：中低 ⚡ 啟動速度：快（2–3 個月可驗證）**

## 機會二：汽車電子 / EV 零組件代理

### 是什麼？
代理台灣車用電子零組件（EV 充電設備、車用感測器、電控模組、電池管理系統）進入 Gurugram/Manesar 汽車產業聚落，服務 Maruti Suzuki、Hero MotoCorp、Ola Electric 等客戶。

### 為什麼現在是好時機？
- 印度是全球第三大汽車市場，2024 年銷售 **420 萬輛**
- 電動車滲透率從 2% 快速成長，2030 年目標 30%（政策強力支持）
- Gurugram 是汽車供應鏈重鎮，距離 Delhi 僅 30 分鐘
- 印度本土車用電子製造能力弱，高度依賴進口，台灣是最佳替代來源

### 市場規模
| 指標 | 數字 |
|------|------|
| 印度汽車零組件市場（2024） | **$670 億美元** |
| 2030 年預估 | **$1,400 億美元** |
| 車用電子佔零組件市場比例 | 約 40%，且持續上升 |
| 印度 EV 市場規模（2030 預估） | **$1,000 億美元** |
| 台灣車用電子出口潛力（目前未進入印度的比例） | 估計超過 90% |

### 商業模式與財務預估
```
收入模式：代理佣金（5–15%）+ 系統整合服務費

保守情境（第 1–2 年）：
  代理 3 個台灣品牌（如 EV 充電零組件）
  年銷售額 $100 萬美元 × 佣金 8% = $8 萬美元

成長情境（第 3–5 年）：
  成為 2–3 個台灣品牌的印度獨家代理
  年銷售額 $500 萬美元 × 佣金 10% = $50 萬美元
  + 技術服務收入 $10 萬美元
  = 總計 $60 萬美元（約 NT$1,920 萬）

長期出口（第 5 年後）：
  建立本土品牌，考慮 OEM 或在印度本地組裝
  利潤率可提升至 20–30%
```

### 你朋友需要做什麼？
1. 加入 Society of Indian Automobile Manufacturers（SIAM）的活動建立人脈
2. 拜訪 Gurugram 的 Maruti 供應商園區，找採購/技術部門
3. 確定 1–2 個具體品類（不要太廣），先做深

### 風險
- 汽車供應鏈認證週期長（IATF 16949 認證）
- 印度本土有 Tata、Mahindra 自己在發展車電，有競爭
- 需要技術背景或合夥人

**風險評級：中 ⚡ 啟動速度：中（6–12 個月見成效）**

## 機會三：PLI 政策顧問 / 台印投資橋接

### 是什麼？
協助台灣廠商（尤其是半導體、電子、醫療器材）申請印度 PLI 補貼計畫，並在當地找到合規的印度合作夥伴，賺取顧問費與成功報酬。

### 為什麼現在是好時機？
- 印度政府砸下 **$260 億美元**的 PLI 資金補貼 14 個產業
- 半導體 PLI 單一項目就有 **$100 億美元**補貼
- 台灣廠商想進入但不懂印度法規、找不到可信的本地合作夥伴
- 你朋友在 Delhi **靠近政府部會**，這是最大差異化優勢

### 市場規模
| PLI 計畫 | 補貼規模 |
|---------|---------|
| 半導體與顯示器 | $100 億美元 |
| 電子製造（手機等） | $18 億美元 |
| 醫療器材 | $2.6 億美元 |
| 白色家電 | $7.5 億美元 |
| **合計** | **超過 $260 億美元** |

顧問市場：每個 PLI 申請案件，外部顧問費約 **$5 萬–$20 萬美元**；若協助台灣廠商在印度設立合資公司，成功報酬可達投資額的 **1–3%**（即 $1,000 萬美元投資 = $10–30 萬美元報酬）

### 商業模式與財務預估
```
收入模式：顧問月費 + 成功報酬

啟動期（第 1 年）：
  2–3 個顧問案 × $5 萬美元 = $10–15 萬美元
  （主要是市場調研、法規梳理、合作夥伴媒合）

成長期（第 2–3 年）：
  5–8 個案子 × 平均 $10 萬美元 = $50–80 萬美元
  + 若促成台廠設廠合資，成功費 $20–50 萬美元/案

長期品牌價值：
  成為「台印投資橋梁」精品顧問公司，
  可募資設立台印產業基金（$1,000–5,000 萬美元規模）
```

### 你朋友需要做什麼？
1. 主動拜訪 DPIIT（工業促進部）、MeitY（電子部）建立政府關係
2. 加入 FICCI 或 CII（印度最大商業組織），快速擴展企業人脈
3. 研究 1–2 個已申請 PLI 的台灣公司案例，理解痛點

### 風險
- 政府關係需要長時間建立，無法快速變現
- 印度官僚流程複雜，客戶耐心有限
- 競爭對手：德勤、麥肯錫等大型顧問公司也在搶這塊

**風險評級：中高 ⚡ 啟動速度：慢（12–18 個月才見收入）**

## 三個方向比較總覽

| | 機會一：電子供應鏈 | 機會二：汽車電子 | 機會三：PLI 顧問 |
|--|----------------|--------------|--------------|
| **啟動資金** | $1–2 萬美元 | $3–5 萬美元 | $2–3 萬美元 |
| **見到收入時間** | 3–6 個月 | 6–12 個月 | 12–18 個月 |
| **第 3 年收入潛力** | $30–60 萬美元 | $40–70 萬美元 | $50–100 萬美元 |
| **你在台灣的角色** | 找供應商、對接報價 | 找品牌授權、技術支援 | 找台灣客戶、協助盡職調查 |
| **朋友在 Delhi 的角色** | 拜訪採購、維護關係 | 代理銷售、技術服務 | 政府關係、法規諮詢 |
| **最大風險** | 被廠商直接繞過 | 認證週期長 | 收入不穩定 |
| **最大機會** | 量大、複製性強 | EV 趨勢強勁 | 單案金額高 |

## 建議啟動順序

```
第 1–3 個月：驗證期
  ✅ 機會一優先啟動（成本最低、最快能驗證）
  → 你聯繫 5 家台灣零組件廠
  → 朋友拜訪 Dixon/Amber 採購部門
  → 看能不能促成第一筆詢價

第 4–6 個月：擴張期
  ✅ 若機會一有進展，同步開始機會二
  → 鎖定 EV 充電零組件這個最熱的細分市場
  → 找台灣有 EV 產品線的廠商談代理

第 7–12 個月：升級期
  ✅ 用前兩個方向建立的資源與人脈，
     開始做機會三（此時你們已有真實案例背書）
```
