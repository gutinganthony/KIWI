## 1. 數據與方法（可重現 + 已驗真）

**數據來源（全部 GitHub 公開、真實）**：
- VIX 日線 1990–2026：`datasets/finance-vix`（源 CBOE）
- CNN Fear & Greed 日線 2011–2026：`whit3rabbit/fear-greed-data`
- S&P500 日線 1990–2021：`fja05680/dow-sp500-100-years` + `investpy` + `VOO` 接合（**含 2000/2008/2020 三大崩盤**）
- TAIEX 日線 2011–2022：`VickykciV/Stock-Market-Network`（含 2020 COVID）

**數據真實性驗證**（對照已知歷史值，全中）：
| 檢查 | 程式讀到 | 已知值 | ✓ |
|---|---|---|---|
| VIX 2020-03-16 | 82.69 | 史上收盤最高 82.69 | ✓ |
| VIX 2008-11-20 | 80.86 | GFC 80.86 | ✓ |
| VIX 2018-02-05 | 37.32 | Volmageddon 37.32 | ✓ |
| F&G 2020-03-23 | 5.0 | COVID 底，極度恐懼個位數 | ✓ |

**方法**：
1. 每指標定義「抄底觸發」(VIX≥25/30/35/40；F&G≤25/20/15/10；RSI≤35/30/25；回檔≤−10/−15/−20%)，200DMA 當牛熊過濾。
2. 窮舉 單/雙/三 條件 AND 組合。
3. 事件**去叢集**（相鄰≥21交易日，避免一段崩盤算成幾十次）。
4. 算 +21/63/126/252 日後報酬；勝率=正報酬比例。
5. **期間配對基準**：每組合跟「同期間無條件勝率」比（避免拿 2011-21 牛市子期間和長期基準比），用**二項檢定**算 p 值。
6. 篩 p<0.05 + 樣本門檻 → 依勝率排序。

> ⚠️ **誠實警告（必讀）**：(a) 窮舉多組合有**多重比較**問題，故我只採信「樣本夠大 + 跨 gap/分半樣本仍穩」的組合（下方旗艦兩個都通過）。(b) 美股 1990–2021、台股 2011–2022，**不含 2022–2026**。(c) **台股 100% 是「2011–21 結構多頭」條件下的結果**——每次深跌一年內都創高才有 100%；**若進入結構性空頭（如台股 1990 或 2000–08 失落十年），此訊號會失效**。真正的風險是「多頭有沒有斷」。

## 3. 美股旗艦組合：`VIX≥30 + 站上200DMA`（逐事件驗證）

**16 事件，6 個月後 100% 正報酬，平均 +15.3%，最差也 +1.0%**：
```
1997-10:+25 1997-12:+21 1998-08:+19 1998-09:+21 1998-10:+26 1998-12:+17
1999-01:+17 1999-02:+10 2000-05:+1  2009-06:+16 2009-10:+13 2010-05:+6
2018-02:+8（Volmageddon） 2020-03:+14 2020-06:+21 2020-09:+11（COVID）
```
**穩健性**：
- 去叢集 gap=10→20 事件/100%；gap=21→16/100%；gap=42→12/100%（不因參數而變）
- **分半樣本**：1990–2004 → 9/9=100%（+17.4%）；2005–2021 → 7/7=100%（+12.6%）

**為何 100%？** 因為 `站上200DMA` 這個過濾把 **2008、2000–02 那種「跌破趨勢的空頭 VIX 暴衝」剔除了**（那時 VIX>30 但價在 200DMA 下 → 不觸發）。剩下的 VIX>30 都是「多頭中的恐慌」→ 歷史上全部收復。**這就是 VIX 單獨用會吃 −40%、加 200DMA 後變 100% 的原因。**

## 5. 怎麼用：抄底檢查表（含現在的狀態）

### 🇺🇸 美股抄底 SOP
```
扣板機條件（全部成立才算「armed」）：
  □ VIX ≥ 30          ← 現在 21.5  ✗（未到）

**現在狀態 = 觀察區**：站對了 regime（多頭），但恐懼還不夠深（VIX 21.5、F&G 42）。**VIX 推上 30（且仍站200DMA）就是歷史最高勝率的扣板機點。**

### 🇹🇼 台股抄底 SOP
```
扣板機條件：
  □ TAIEX 距 52 週高點回檔 ≥ 10%（更強：≥15%）  ← 現在約 −1.7%  ✗（太淺）
  → 成立 = 歷史 12 個月 100% 勝率、平均 +18%
  → 持有期：必須抱滿 12 個月（3 個月只有 72%，別短打）
  ⚠️ 前提：台股結構多頭未斷（TSMC/AI 供應鏈故事還在）。若多頭斷，此訊號失效。

### 分批用法（綁回前一份 roadmap 的子彈）

## 7. 統計嚴謹度檢驗（反覆驗證的核心）

跑 `--robust`：① **去重複**（同訊號組合收斂成一個，避免多重比較母數灌水）② **BH-FDR 多重比較校正** ③ **樣本外**（train/test 分期）④ **Wilson 95% 下界**。

**兩階段驗證（重點：延長資料把結論「翻轉」了——這正是嚴謹的價值）**

| 階段 | 美股最硬組合 | 台股最硬組合 |
|---|---|---|
| **資料只到 2021/2022** | VIX≥30+200DMA：樣本外穩但 **過不了 FDR**（n16、測167組） | 回檔≥10%+跌破200DMA：**過 FDR**（但100%是2011–21多頭虛胖） |
| **補到 2026（含22空頭/23/25）** | **回檔≥10%+站上200DMA：✅ 過 FDR**（n=37、95%、Wilson82%、樣本外10/100%） | 家族 18 條全 80–89% **但無人單獨過 FDR**（納入2000/2008後回到誠實值） |

**白話結論（誠實版）**：
1. **補到 2026 後，最硬的變成美股**：`回檔≥10% + 站上200DMA`（抱6月）n=37、95%、Wilson 下界 82%、樣本外（含2020/2022）10次100%、**138組唯一過 FDR**。**這條可當交易紀律。**
2. **台股的 100% 是假象、被資料修正了**：之前只有 2011–21（沒2000/2008），所以 100%；補進兩大空頭後回到 **80–89%**——更可信但達不到單一 FDR 過關。**用「家族一致性」看：18 個深跌變體全 80–89% 同向，可信，但別宣稱單一鐵律。**
3. **Fear & Greed 實測落榜**：F&G≤10/15/20/25 都測了，受限 2011+ 樣本 + 3M edge 弱，沒進顯著榜。**贏家是「價格回檔深度 + 200DMA 趨勢」，不是恐懼指數。**
4. **方法論收穫**：把資料從 2021 補到 2026，**美股訊號升級成鐵證、台股虛胖被打回原形**——證明「反覆驗證 + 擴大樣本」確實會改變結論，不能只看一段行情。
