# AVI V4 市場風險指標 — 完整介紹指南

> **AVI（Adjusted Valuation Index）V4** 是一個綜合市場風險評分系統，將 **14 個獨立的金融指標**、跨越 **6 個維度**，整合為一個 **0–10 分** 的單一分數。分數越高 = 市場風險越大。

*最後更新：2026-04-20 ｜ 當前版本：V4 ｜ 當前 AVI：**6.73** 🟡 偏高風險*

---

## 一、V4 為什麼存在？——V3 的三個無法迴避的問題

V3 在 2023 年設計時表現很好，但到 2026 年撞上三個結構性問題：

1. **資料源失效**
   - STLFSI2（聖路易聯儲金融壓力指數）於 2022-01 **停止發布**
   - FRED 自 2026-04 起把 HY/IG 信用利差（BAMLH0A0HYM2 / BAMLC0A0CM）**限制為僅 3 年歷史**
   - V3 的信用維度和部分宏觀維度在 2022 後實質上「瞎了」

2. **CAPE 結構性飽和**
   V3 用的是「擴展窗口百分位」（把當前 CAPE 對比從 1990 以來的所有值）。但 2020 年後 CAPE 長期在 30 以上，用擴展窗口永遠顯示 90%+ 百分位，失去區分度。CAPE=30 還是 CAPE=40 對模型而言都是「極高」。

3. **2022 升息空頭漏報**
   V3 完全沒有「通膨／實質利率」獨立信號。2022 年股市從 1 月崩盤到 10 月、CPI YoY 達 9%，但 V3 在 2022 高峰前最高分僅 6.95，剛好邊緣。沒有利率維度，這類「利率驅動」的修正幾乎無法被捕捉。

V4 針對這三個問題逐一解決。

## 二、V4 三大升級

### 升級 1：新增「利率 / 通膨」維度（A）
加入兩個獨立指標：

- **CPI YoY %**（消費者物價指數年增率）— 直接量測通膨熱度
- **10Y Real Yield**（10 年期實質利率，用 `DGS10 − CPI YoY` 算 ex-post 實質利率）— 量測貨幣政策緊度

**為什麼有用**：2022 年 1 月 CPI YoY 衝到 7.56%（歷史 100 百分位），實質利率轉正到數十年新高 — V4 在該時點的利率維度直接觸發警報，補齊 V3 的盲點。

### 升級 2：改用「20 年滾動百分位」方法（B）
每個指標不再對比「從 1990 以來的全部歷史」，而是**對比過去 240 個月（20 年）的數據**。

**為什麼有用**：
- 解決 CAPE 結構性飽和 — 2026 年 CAPE 40.44 在 20 年滾動窗口內仍是 97 百分位，但「97%」是相對於 2006 年以來的範圍判定，對當前週期更有參考性
- 適應結構性轉變（升息週期切換、AI 時代估值重訂等）
- 保留足夠歷史（20 年 > 一個商業週期），統計穩定性不差

### 升級 3：長歷史替代的信用指標
V3 的 HY / IG OAS 已無法取得完整歷史。V4 改用：

- **BAA-10Y Spread**（Moody's Baa 公司債 − 10 年期國債）— 資料從 **1953** 起
- **BAA-AAA Differential**（Baa − Aaa 品質差異利差）— 資料從 **1919** 起

兩者都是信用風險的經典長歷史代理，在每次危機（2000、2007、2020）都同步或提前擴大。

### 另外修掉的：
- STLFSI2 → **STLFSI4**（聖路易聯儲同系列的替代版本，仍持續更新）
- CAPE 資料來源改以 multpl.com 為主（Yale XLS 作 fallback）— 確保能拉到最新資料

## 三、AVI 分數怎麼看

| 分數區間 | 燈號 | 含義 | 建議行動 |
|----------|------|------|----------|
| 8.0–10.0 | 🔴 極度危險 | 極度高估，崩盤風險極高 | 大幅降低風險敞口、提高現金 |
| 7.0–8.0 | 🟠 高風險 | 顯著高估 | 減倉、避免追高 |
| 6.0–7.0 | 🟡 偏高 | 偏貴，需謹慎 | 謹慎操作、不加倉 |
| 5.0–6.0 | 🟡 中等偏高 | 略高於歷史中位數 | 正常配置、留意變化 |
| 4.0–5.0 | 🟢 中性 | 估值合理 | 正常投資配置 |
| 3.0–4.0 | 🟢 偏低 | 偏便宜 | 潛在買入機會 |
| 0.0–3.0 | 🟢 極低 | 歷史性低估 | 積極佈局 |

**核心觀念**：
- **趨勢比絕對值更重要** — AVI 從 5.5 連漲到 6.3 比穩定在 6.5 更值得擔心
- **看維度結構，不只看總分** — AVI 6.5 如果來自估值單邊偏高（其他維度都低），和來自估值+信用+宏觀同時走高，是完全不同的風險結構

## 四、計算邏輯（3 步驟）

### 步驟 1：蒐集 14 個指標當月數值
每月月初 resample，14 個指標的最新月度值。

### 步驟 2：算「過去 20 年滾動百分位」
對每個指標 `x(t)`，取 `t-240` 到 `t-1` 這 240 個月的歷史值，計算 `x(t)` 在此分佈中的排名百分位：

```
pctile(t) = rank(x(t), history[t-240 : t-1]) / 240 × 100
```

- 對「越高 = 越危險」的指標（如 CAPE、VIX、CPI），pctile 直接轉為 0-10 分：`score = pctile / 10`
- 對「越高 = 越安全」的指標（如 FCF Yield、Drawdown），反轉：`score = (100 − pctile) / 10`

### 步驟 3：加權求和
```
AVI = Σ (指標評分 × 權重)
```

## 六、權重分配邏輯

```
估值 38% ━━━━━━━━━━━━━━━━━━━━━━━━━━━━ ← 核心，泡沫本質
獲利  8% ━━━━━━━━                        ← 估值需獲利驗證
宏觀 14% ━━━━━━━━━━━━━                   ← 外部環境
利率 14% ━━━━━━━━━━━━━                   ← ⭐ V4 新增，政策緊縮壓力
信用 12% ━━━━━━━━━━━                     ← 前哨站，先於股價下跌
動量 14% ━━━━━━━━━━━━━                   ← 平衡器，健康多頭不假警
```

**V4 相對 V3 的權重變化**：
- 估值從 42% → 38%（讓出權重給利率）
- 獲利從 10% → 8%（單一指標適當縮減）
- 宏觀從 18% → 14%（Yield Spread 貢獻縮減）
- 新增利率 14%
- 信用從 14% → 12%
- 動量從 16% → 14%

設計邏輯：V3 讓估值獨大（42%），在 AI 時代估值長期偏高的背景下會產生「永遠亮紅燈」的問題。V4 把權重分散到新加的利率維度和長歷史信用指標，讓信號更均衡。

## 九、使用場景

### 場景 1：每月風險監控
- 每月月初查 AVI 分數與趨勢
- AVI 連續 3 個月上升 → 開始留意
- AVI 從 5.5 以下升到 6.0 以上 → 降低高風險資產比例
- AVI 突破 7.0 → 認真考慮大幅減倉

### 場景 2：逢低買入的信心指標
AVI 低於 4.0 通常代表大幅修正後的低估區域。歷史上（2003、2009、2020 年 3 月底），AVI 低於 4.0 後 12 個月市場表現通常優於平均。

### 場景 3：多維度風險拆解
同樣 AVI = 6.5，結構不同意義不同：
- 估值 97% + 其他 10%：**單邊估值泡沫**，可能 CAPE 壓縮、但盈利成長可消化
- 估值 60% + 信用 70% + 宏觀 80%：**多重壓力同時出現**，更危險

### 場景 4：和其他分析工具搭配
AVI 不取代基本面或技術分析，而是提供「市場體質背景」。個股分析叫你買 + AVI 說環境偏危險 = 少買一點或等時機。

## 十、限制與注意事項

| 項目 | 說明 |
|------|------|
| **不是擇時工具** | 告訴你「環境危險」，不是「明天崩盤」。崩盤可能在 AVI 亮紅後 1 個月或 1 年發生 |
| **趨勢 > 數值** | AVI 6.5 穩定 3 個月 vs AVI 6.0 連漲 3 個月 — 後者更危險 |
| **無法預測黑天鵝** | COVID 類衝擊無法事前預測，但 V4 告訴你「如出事，市場有無安全邊際」 |
| **資料延遲** | CPI / BAA 等月度數據，有 1 個月以內的延遲 |
| **20Y Rolling 的代價** | 前 20 年歷史（1990-1995）AVI 會返回 NaN，因為窗口還沒累積足夠 |
| **歷史不等於未來** | 5 次危機回測表現好，不保證未來必定有效。**V4 是風險監控工具，不是水晶球** |
