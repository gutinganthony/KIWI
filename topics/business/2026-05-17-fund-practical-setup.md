## 你的做法

朋友開元大證券帳戶 → 帳密給你操作 → 資金綁定朋友本人帳戶 → 朋友隨時可登入查看。

### 為什麼這樣可行

| 優點 | 說明 |
|------|------|
| 資金不可能外流 | 券商出金只能到本人銀行帳戶，你拿帳密也轉不走錢 |
| 朋友隨時監控 | 同一組帳密，對方隨時 APP 看得到持倉和損益 |
| 零成本 | 不需要 IBKR 的 Advisor 架構，不需要任何額外帳戶 |
| 操作簡單 | 就是一般下單，不需要學新系統 |

### 需要注意的風險

| 風險 | 說明 | 解法 |
|------|------|------|
| **同時登入衝突** | 你在下單時朋友也登入操作 → 可能衝突 | 口頭約定：開盤時間帳戶由你操作，對方只看不動 |
| **對方自己亂買** | 朋友看到某檔股票心癢自己下單 | 合約寫清楚：授權期間對方不自行交易，否則盈虧自負 |
| **密碼被改** | 朋友改密碼沒通知你 | 口頭約定即可，反正改密碼 = 終止合作 |
| **多帳戶管理效率** | 5 個人 = 5 組帳密，每次操作要登入登出 5 次 | 用不同瀏覽器 profile 同時登入 |

### 多帳戶效率解法

```
Chrome Profile 1 → 朋友 A 的元大帳戶
Chrome Profile 2 → 朋友 B 的元大帳戶  
Chrome Profile 3 → 朋友 C 的元大帳戶
Safari          → 你自己的帳戶

全部同時開著，一次下單完畢。
或者用元大的 APP 在手機 + iPad + 電腦分開登入。
```

### 操作覆蓋範圍

| 市場 | 元大證券 | 備註 |
|------|---------|------|
| 台股 | ✅ | 直接交易 |
| 美股 | ✅ | 元大複委託（手續費較高，約 0.5-1%）|
| 港股 | ✅ | 元大複委託 |
| 日股 | ✅ | 元大複委託 |

如果美股交易量大，可以請朋友額外開一個**海外券商**（如 Firstrade、嘉信）專門做美股，手續費更低。台股用元大，美股用海外券商，兩套帳密分開管理。

## 法律風險評估（更新版）

### 你的情境

- 2-3 位朋友
- 用對方的券商帳戶（帳密共享）
- 資金綁定對方本人帳戶
- 預期虧損幅度 10-20%
- 不公開招攬

### 最糟情境試算（朋友翻臉 + 檢舉）

```
假設朋友投入 500 萬，虧損 20% = 虧 100 萬

刑事（機率 < 5%）：
  緩刑 2 年 或 易科罰金 54 萬
  罰金 100 萬（法定最低）

民事（機率 < 10%）：
  有合約：賠 1 倍 = 100 萬
  沒合約：賠 2-3 倍 = 200-300 萬

最糟總花費：
  有合約：律師 30 萬 + 罰金 100 萬 + 民事 100 萬 = 230 萬
  沒合約：律師 30 萬 + 罰金 100 萬 + 民事 300 萬 = 430 萬

→ 有合約省 200 萬
```

### 機率總結

| 情境 | 機率 | 備註 |
|------|------|------|
| 一切順利，什麼事都沒有 | **85-90%** | 最大機率 |
| 虧錢但朋友接受，關係微妙 | **5-10%** | 需要好的溝通 |
| 翻臉但只吵架不告 | **3-5%** | 大多數人不會走法律程序 |
| 翻臉 + 民事求償 | **1-3%** | 有合約 = 少賠很多 |
| 翻臉 + 刑事檢舉 | **< 1%** | 極端情況 |

## 你現在該做的 Checklist

- [ ] 跟朋友面對面談風險承受度（用情境題）
- [ ] 確認每人只投淨資產 20% 以下
- [ ] 請朋友開好元大證券帳戶
- [ ] 拿到帳密，測試登入
- [ ] 用 LINE 發送確認訊息（白話文，不用專業術語）：
  ```
  確認一下我們的合作方式：
  1. 你把帳戶交給我幫忙操作
  2. 投資有風險，可能會虧
  3. 你隨時可以收回
  4. 我每月發對帳給你
  5. 費用的部分：每年收帳戶金額的1.2%當管理費，
     如果年報酬超過8%，超過的部分我抽2成
  6. 每季結算一次，我會算給你看
  確認的話回我👍
  ```
  ⚠️ 重點：用白話文不用專業術語。
  「管理費率 1.2% p.a.」→「每年收帳戶金額的 1.2%」
  「Performance fee 20% above hurdle」→「超過的部分我抽 2 成」
  看起來像朋友聊天，不像在經營業務。
- [ ] 設定好 Chrome 多 Profile 登入
- [ ] 開始操作
- [ ] 每月 5 日發月報
