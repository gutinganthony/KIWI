## Summary

基於 NVIDIA 財報揭露 Vera CPU 預期年銷售 $200 億，推算 SoCAMM 記憶體的需求量。結論：**單單 NVIDIA 的 SoCAMM 需求，就會把三大廠 2026-2027H1 新增的常規 DRAM 產能全部吃光，造成月均 1.1-3.1 萬片晶圓的淨缺口。** 中國 CXMT 在 2027 年還無法形成足以衝擊市場的產出。

---

## 一、需求推算

### KIS（韓國投資證券）版本

| 項目 | 數字 |
|------|------|
| Vera CPU 年銷售額 | $200 億 |
| Grace CPU ASP 估計 | $3,000-$5,000 |
| Vera CPU ASP 估計（更高） | $5,000-$8,000 |
| 推算出貨量 | **250-400 萬顆** |
| 每顆 SoCAMM 插槽 | 8 個 |
| 每槽容量 | 192GB-256GB |
| 每顆插滿容量 | **1.5TB** |
| SoCAMM 總需求（保守 250 萬顆 × 1.5TB）| **300 億 Gb** |

### 石槌夫修正版（更保守）

| 項目 | 修正 |
|------|------|
| Vera 數量 | 250 萬顆（同意 KIS 保守值）|
| 每顆實際用量 | **768GB**（1.5TB 的一半，不是所有都插滿）|
| SoCAMM 總需求 | **150 億 Gb**（年增 15 倍以上）|

## 二、供需缺口計算

### SoCAMM 晶圓需求

```
bit 需求：250萬 × 768GB × 8 bits = 15,360 Billion Gb
理想晶圓：15,360B Gb ÷ 24,000 Gb/片 = 640,000 片/年
月均：53,300 片/月
良率修正（÷ 0.75）：71,000 片/月

CY2025 基準：約 700 片/月

→ CY2026 SoCAMM 淨增需求：~70,300 片/月
```

### 三大廠 CY2026 新增常規 DRAM 供給

| 廠商 | 新增產能 | 常規 DRAM 淨增 | 備註 |
|------|---------|---------------|------|
| **Samsung** | P4 擴建 +2-3 萬片 | **+2-3 萬片/月** | HBM 17→25 萬片壓縮常規。P5 要 2028H2 才量產 |
| **SK Hynix** | M15X 擴至 7 萬片 | **+2-3 萬片/月** | 60% 分配 HBM。Y1 廠 2027/2 才運營 |
| **Micron** | ID1 要 2027 中才出 | **~0** | 靠 1β→1γ 微縮增 20%，非新增晶圓 |
| **合計** | | **+4-6 萬片/月** | |

### 淨缺口

```
SoCAMM 需求：+7.1 萬片/月
三大廠供給：+4-6 萬片/月

淨缺口：-1.1 至 -3.1 萬片/月 ❌

注意：這還沒計算 Google TPU、Amazon 等其他需求
```

## 三、2027H2-2028 產能展望

### 各廠新增常規 DRAM

| 廠商 | 2027H2 | 2028 | 備註 |
|------|--------|------|------|
| Samsung | ~0 | +1.5-3 萬 | P5 設備安裝中，2028 才爬坡 |
| SK Hynix | +1.5-3 萬 | +3-5 萬 | Y1 爬坡初期 |
| Micron | +1-2 萬 | +4-7 萬 | ID1 + P5 逐步滿載 |
| CXMT | +1-3 萬 | +3-5 萬 | 上海新廠，受設備限制 |
| **合計** | **+3.5-8 萬** | **+12-20 萬** | |

### 同期 SoCAMM 需求

| 時間 | 需求 | 驅動力 |
|------|------|--------|
| 2027H2 | **10-14 萬片/月**（CY2026 的 1.5-2 倍）| Vera Rubin NVL72 全面量產 |
| 2028 | 持續增長（難估算）| 更多 AI 推論需求 |

> **2028 年 NVIDIA 單一客戶的 SoCAMM 需求就可能消耗新增產能的一半。**

## 四、CXMT 能不能救市？

| 限制 | 說明 |
|------|------|
| 設備限制 | 美國出口管制，先進 EUV 設備取得困難 |
| 安裝時間 | 2026H2 開始裝機 → 穩定量產需 6-12 個月 |
| 良率爬坡 | 新廠良率從 50% 爬到 80%+ 需要 6-12 個月 |
| 實際貢獻 | 2027H2 才 1-3 萬片/月，要看 2028 |

> **結論：「中國 2027 下半年造成記憶體大崩盤」= 純屬鬼故事。** 物理限制、設備限制、時間限制都不支持。

### 對照 [HBM 光互連 CPO 分析](./2026-05-17-hbm-optical-interconnect-cpo.md)

- SoCAMM 是「可插拔記憶體」的第一步實現——Jukan 說的概念已經在 Vera CPU 上發生了
- 每顆 CPU 8 個插槽 × 可獨立升級 = 記憶體確實變成可替換元件
- 但需求不會減少，反而因為 AI 推論需求會持續膨脹

### 投資含義

| 結論 | 行動 |
|------|------|
| DRAM 供需 2026-2027H1 會持續緊張 | 記憶體股短期內不用急著跑 |
| SoCAMM 是新的大量需求來源 | 跟蹤 NVIDIA Vera CPU 出貨進度 |
| CXMT 短期內不會砸盤 | 「中國崩盤論」可以忽略 |
| 真正該擔心的是 2028+ | 三大廠 + CXMT 同時放量 + AI capex 是否持續 |
