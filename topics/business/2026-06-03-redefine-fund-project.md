1. **Jake 真正的稀缺資產是履歷，不是 AVI 工具**：國泰家辦團隊在職身份（潛在 LP）、真實募資戰績、台灣/HK 牌照、跨境結構（QDII/MRF/QDLP）。下單交易是最該外包的事，角色要往上提到「資本 + 結構 + 風控疊加」。
2. **AVI / Serenity 是基金的專有研究引擎，不是被丟掉的副業**：
   - Vince = 自下而上（持有什麼、技術護城河）
   - AVI = 自上而下風險 regime（何時加減碼、部位多大）
   - Serenity 卡脖子框架 = 半導體/AI 選股流程（找隱藏壟斷供應商）
3. **redefine = 跑道現金流 + 行銷飛輪**：沒 AUM 的前 6–12 個月，靠發旗艦研究報告建立可信度與受眾、同時賣研究養團隊、產生 LP inbound。先成為「被看見的研究機構」，再轉基金（Leopold 模式的真正機制）。
4. **台灣地利 = Vince 的 ASML 校友網（合法 channel check / 專家網路）**，要變成可重複研究 SOP，並嚴守 MNPI / 內線紅線。

### 關於「在職 + 海外」的合規地圖（重要）
**地點不解決衝突，觸發紅線的是「角色」**：是否管別人的錢並收費、是否在受規管實體掛登記角色/持股、是否招攬客戶、國泰合約的兼職/競業/申報條款（多半不分地域）。
→ **乾淨路徑**：Phase 1 由 **Vince 當檯面上持牌主體 + 研究官**先建實體；Jake 檯面下用自有資金共同注資 + 提供研究，**不掛受規管角色、不收費、不碰客戶**；要收外部錢時 Jake 再有序退場轉正式角色。
- ✅ 自有資金 IBKR 投資 = 個人投資，可行。
- 🟡 與 Vince 共同注資跑平行帳戶 = 需保持「自有資金、零外部 LP、零收費、Jake 不掛角色」。
- 🔴 Jake 在職期間於 HK EAM 當持牌 RO = 明顯衝突，通常過不了 → Vince 先持牌。
- 🔴 向 LP 募資收費 = 離職後才做。
- ⚠️ 必做：讀國泰合約利益衝突/兼職條款 + 諮詢台灣勞動法&證券法律師（確認持有海外實體股份是否需申報）。本檔為實務地圖，非法律意見。

## 7. 補充（2026-06-03 pt.2）：國泰合約結論 + 結構 step-by-step + 離職時點

### 7.1 國泰合約（錄取通知書）關鍵結論
- **第十一條（特別約定）**：「非經董事長核准，**不得兼任本行以外之其他職務**（含公司董事、監察人或其他兼職）」。第九條：違反工作規則者本行得**立即停職或終止契約**。
- → 在職期間 **Jake 不能掛任何基金正式角色**（GP/董事/RO/負責人）。Vince 先當持牌主體，Jake 僅自有資金投資 + 研究、不掛角色、不收費、不碰客戶。
- 錄取通知書**未見離職後競業禁止**條款（台灣離職後競業禁止須另簽 + 給補償才有效，上限 2 年）→ 離職後做基金，從此文件看無限制。
- ⚠️ **待補洞**：第十一/十三條「依本行**工作規則**」辦理 → 真正的個人帳戶交易申報/利益衝突/外部持股規範在工作規則與法遵手冊。**行動：索取 ①工作規則 ②員工自營買賣規範 ③利益衝突/外部活動申報政策**，確認「IBKR 自營」與「被動持有海外資管公司股份」是否需事前申報。看到前先別持股。
- **底薪：NT$132,000/月**（專業經理 / 私人銀行資深專屬投資組合顧問，2025-07-30 報到）。年化底薪 ~NT$158萬；總包估 ~NT$200–280萬（「一般般」）。

### 7.2 結構 step-by-step（破除混淆）
- **牌照層**（誰能合法管錢）：HK Type 9 / SG CMS。**外殼層**（錢裝哪）：開曼基金 / SG VCC。**開曼是外殼不是牌照**，仍需 HK/SG 經理人。
- **HK EAM (Type 9)**：註冊 HK 公司 → 申 Type 9（需 ≥2 RO、≥3 年資歷、FRR、法遵；審 6–12 月）→ 簽 IMA（錢留客戶私銀）。痛點：Vince 無持牌資歷、Jake 在職不能掛 → **捷徑：掛靠現成持牌平台/孵化器當 sub-manager（數週-數月、資本最輕）= 最快真實起點**。
- **SG (CMS+VCC)**：註冊 SG 公司 → CMS 牌照（S$250k 資本、本地實質、MAS 審 3–6 月）→ VCC 外殼（可申 13O/13U 稅優）。痛點：本地實質、你人不在、門檻高。
- **開曼 feeder**：定 master-feeder → 開曼律師/行政/審計/PPM → CIMA 註冊 → 指派 HK/SG IM（仍需持牌）。設立 US$30–80k+、年費 US$30–60k+，**需承諾資本 ≥US$10–20M 才划算**。
- **推薦階梯**：① 自有資金 IBKR 練績效 → ② 掛靠 HK 持牌 EAM 平台（錢留私銀、Vince 持牌）→ 規模化後自建 HK Type 9 →（主攻國際/美元）再加開曼 feeder 或 SG VCC。別一開始碰開曼/SG。

### 7.3 離職時點 / break-even AUM
- 模型：**需要 AUM ＝（目標年收入 ÷ 分潤比 ＋ 年營運成本）÷ 管理費率**；假設分 50% 管理費淨額、年成本 ~NT$250萬。
- **「費收取代現職」甜蜜點 ≈ 委管 AUM US$10–15M（NT$3–5億）**：追平底薪 ~US$9–12M、追平總包 ~US$11–15M。
- 真正財富是 **carry**（US$15M×15%報酬×15%carry ≈ Jake 分 NT$540萬/好年度）但浮動、不能當辭職依據。
- **離職四閘門（全綠才走，估 18–30 個月）**：①房貸已撥款 ②track record ≥12–18 月真錢可驗證 ③committed AUM ≥US$10M（含 anchor）+ 12 月內看到 US$15M pipeline ④結構/牌照 ready。
- **推薦扣板機：committed AUM 觸及 ~US$12M（NT$3.8億）**。在那之前留國泰養房貸+緩衝金（建議存 12–18 月生活費 ~NT$250–350萬）。
