## 二、成本與淨收入分析

### 不成立公司（Friends & Family 階段）

| 項目 | 年成本 |
|------|--------|
| IBKR 交易佣金 | ~$2-5K（依交易頻率）|
| 數據服務（FRED 免費、yfinance 免費）| $0 |
| Bloomberg / Refinitiv（可選）| $0-27K |
| 會計/稅務（個人申報）| ~$1-2K |
| **總成本** | **~$3-7K / 年（10-22 萬 TWD）** |

```
AUM $3M、報酬 18% 的淨收入：
  毛收入：$96K（307 萬 TWD）
  成本：-$5K（16 萬）
  稅前淨收入：$91K（291 萬 TWD）
  個人所得稅 (~20%): -$18K
  ────────────────────
  稅後淨收入：~$73K（234 萬 TWD）
```

### 成立公司後的成本結構

| 項目 | 年成本（TWD） | 備註 |
|------|-------------|------|
| 公司設立（首年）| 5-10 萬 | 會計師、登記費 |
| 記帳 + 稅務申報 | 3-6 萬/年 | 會計事務所 |
| 辦公室（可選）| 0-24 萬/年 | 共享空間或在家 |
| 營業稅 + 營所稅 | 營收的 ~20% | 營所稅 20% |
| 勞健保 | 3-5 萬/年 | 負責人自付 |
| 法律顧問（可選）| 2-5 萬/年 | 合約審核 |
| 合規成本 | 0-10 萬/年 | 視是否申請投顧牌照 |
| **總成本** | **12-60 萬/年** | 依規模而定 |

## 三、成立公司時機判斷

### 決策矩陣

| AUM | 人數 | 年營收(18%) | 建議 | 原因 |
|-----|------|-----------|------|------|
| < $500K | 1-3 | < 51 萬 | ❌ 不需要 | IBKR F&F 足夠，成本不划算 |
| $500K-$2M | 3-8 | 51-205 萬 | 🟡 可選 | 考慮寫正式合約保護自己 |
| **$2M-$5M** | 8-15 | **205-512 萬** | **✅ 建議成立** | IBKR 15 人上限、稅務規劃、法律保護 |
| > $5M | 15+ | > 512 萬 | ✅ 必須 | 超過 F&F 限制、需要專業架構 |
| > $10M | 20+ | > 1024 萬 | ✅ 考慮投顧牌照 | 規模夠大，牌照帶來信任 |

### 成立公司的三個觸發點

1. **IBKR 限制**：超過 15 個子帳戶或 5 個持有人 → 必須升級
2. **AUM > $2M**：開始有法律和稅務規劃的價值
3. **年營收 > 200 萬 TWD**：公司架構的節稅效果開始明顯

## 四、募資規劃路徑

### Phase 1：種子期（現在 → 6 個月）

| 目標 | 做法 |
|------|------|
| AUM: $300K-$500K | 2-3 位朋友，每人 $100-200K |
| 建立 track record | 用 TSI/CPI/AVI 系統化管理 |
| 月報機制 | 每月發送績效報告 |
| **營收：30-50 萬/年** | 不辭主業，兼著做 |

### Phase 2：成長期（6-18 個月）

| 目標 | 做法 |
|------|------|
| AUM: $1M-$3M | 口碑擴散，5-10 位客戶 |
| 正式合約 | 請律師擬定投資管理合約 |
| 考慮成立公司 | 當 AUM > $2M 時啟動 |
| **營收：100-300 萬/年** | 可以開始考慮全職 |

### Phase 3：規模期（18-36 個月）

| 目標 | 做法 |
|------|------|
| AUM: $5M-$10M | 15-20 位客戶，開始有大額客戶 |
| 成立公司 | 有限公司或投顧公司 |
| 系統化 | Dashboard + 自動月報 + 風控 |
| **營收：500-1000 萬/年** | 🎯 達標！ |

### Phase 4：專業期（3 年+）

| 目標 | 做法 |
|------|------|
| AUM: $10M+ | 考慮投顧牌照 |
| 團隊化 | 聘請 1 名分析師 + 1 名行政 |
| 產品化 | AVI/CPI/TSI 可以變成付費訂閱服務 |
| **營收：1000 萬+ /年** | 穩定經營 |

## 五、關鍵風險

| 風險 | 影響 | 緩解方式 |
|------|------|---------|
| **虧損年不收績效費** | 營收可能歸零（只剩管理費）| 管理費確保基本收入 |
| **朋友關係破裂** | 虧錢 = 翻臉 | 合約先行、高水位線、定期溝通 |
| **法規風險** | 無照代操可能違規 | AUM < $2M 走 F&F 灰色地帶，大了就考慮牌照 |
| **集中風險** | 1-2 個大客戶佔 80% AUM | 客戶分散化 |
| **績效壓力** | 1-2 季虧損 → 客戶贖回 | 止損規則 + 透明溝通 + 長期思維教育 |
