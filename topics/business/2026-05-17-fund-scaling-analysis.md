## Summary

以個人身份代操朋友資金，全部用台幣計算，不考慮稅務和公司架構。分析從 0 到年毛收入 1000 萬台幣需要多少管理資金（AUM），以及分階段的募資路徑。

---

## 一、收費假設

| 項目 | 數值 |
|------|------|
| 管理費 | 1.2% / 年 |
| 績效費門檻（Hurdle Rate） | 年化 8% |
| 績效費分成 | 超過 8% 的部分，你拿 20% |
| 高水位線 | 有（虧損年不收績效費）|

### 報酬率 25%（樂觀）

| AUM（台幣） | 管理費 | 績效費 | **你的毛收入** |
|------------|--------|--------|--------------|
| 300 萬 | 3.6 萬 | 10.2 萬 | **13.8 萬** |
| 1,000 萬 | 12 萬 | 34 萬 | **46 萬** |
| 3,000 萬 | 36 萬 | 102 萬 | **138 萬** |
| 5,000 萬 | 60 萬 | 170 萬 | **230 萬** |
| 1 億 | 120 萬 | 340 萬 | **460 萬** |
| **2.2 億** | **264 萬** | **748 萬** | **1,012 萬 ✅** |

### 績效費計算說明

```
以 AUM 1 億、報酬率 18% 為例：

年初 AUM：1 億
年底帳戶值：1 億 × 1.18 = 1.18 億（賺了 1,800 萬）

管理費 = 1 億 × 1.2% = 120 萬

績效費 = 超過 8% 的部分 × 20%
       = (18% - 8%) × 1 億 × 20%
       = 10% × 1 億 × 20%
       = 1,000 萬 × 20%
       = 200 萬

你的毛收入 = 120 萬 + 200 萬 = 320 萬
客戶淨賺 = 1,800 萬 - 320 萬 = 1,480 萬（+14.8%）
```

## 三、🎯 年毛收入 1000 萬台幣的目標

| 你的績效 | 所需 AUM | 客戶數（avg 1,000 萬/人） |
|---------|---------|------------------------|
| 保守 12% | **5 億** | ~50 人 |
| **中等 18%** | **3.2 億** | **~32 人** |
| 樂觀 25% | **2.2 億** | ~22 人 |

### 最務實路徑

> **AUM 3 億台幣左右，年化報酬 18-20%，毛收入就能到 1000 萬。**
> 如果平均每位客戶託管 1,500 萬，大約需要 **20 位客戶**。

## 五、分階段募資路徑（全台幣）

### Phase 1：起步（現在 → 6 個月）

| 項目 | 目標 |
|------|------|
| AUM | 1,000-1,500 萬 |
| 客戶數 | 2-3 位朋友 |
| 每人投入 | 300-500 萬 |
| 預期毛收入 | 32-48 萬/年 |
| 你的狀態 | **兼職**（不辭主業）|

重點：建立 track record，每月發績效報告，讓朋友看到系統化管理。

### Phase 2：口碑擴散（6-18 個月）

| 項目 | 目標 |
|------|------|
| AUM | 3,000-5,000 萬 |
| 客戶數 | 5-8 位 |
| 每人投入 | 500-1,000 萬 |
| 預期毛收入 | 96-160 萬/年 |
| 你的狀態 | **兼職但認真考慮全職** |

重點：朋友介紹朋友，track record 累積到 12 個月+，開始正式合約。

### Phase 3：全職投入（18-36 個月）

| 項目 | 目標 |
|------|------|
| AUM | 1-2 億 |
| 客戶數 | 10-15 位 |
| 每人投入 | 800-2,000 萬 |
| 預期毛收入 | 320-640 萬/年 |
| 你的狀態 | **全職**（辭掉主業）|

重點：到這個規模，管理費 120-240 萬/年已經能覆蓋基本生活。

### Phase 4：達標（3 年+）

| 項目 | 目標 |
|------|------|
| AUM | **3-5 億** |
| 客戶數 | 20-35 位 |
| 預期毛收入 | **800-1,600 萬/年** |
| 你的狀態 | **開始考慮成立公司** |

重點：毛收入到 1000 萬 → 正式成立公司做稅務規劃 + 品牌化。

## 七、關鍵數字記憶

- **毛收入 100 萬/年** → AUM 5,000 萬（18% 報酬）
- **毛收入 300 萬/年** → AUM 1 億
- **毛收入 500 萬/年** → AUM 1.5-2 億（可以全職了）
- **毛收入 1,000 萬/年** → AUM 3.2 億 🎯
- **開公司時機** → 毛收入 > 1,000 萬/年 或 客戶 > 15 人
