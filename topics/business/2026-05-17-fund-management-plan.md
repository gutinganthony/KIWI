## Summary

2 位以上朋友想託付資金管理，覆蓋美股、台股、日股、港股。本文設計三個面向：(1) 不成立公司的帳戶管理方式、(2) 合理收費模式、(3) 參考私募的投資管理流程。

## 一、帳戶管理方式（不成立公司）

### 最推薦：Interactive Brokers Friends & Family Advisor

| 項目 | 說明 |
|------|------|
| **方案** | IBKR Friends & Family Advisor Account |
| **限制** | 最多 15 個子帳戶、5 個不同帳戶持有人、AUM < $2M |
| **優勢** | 單一登入管理所有人的帳戶、統一下單、自動分配 |
| **市場覆蓋** | 美股 ✅、港股 ✅、日股 ✅、台股 ❌（需搭配台灣券商） |
| **費用** | IBKR 標準佣金，無額外管理帳戶費 |
| **不需要** | 不需要 RIA 註冊（15 帳戶以下豁免）|
| **設定方式** | 你開 Master Account → 朋友各自開 Sub-Account → 連結到你名下 |

**運作方式：**
1. 你擁有一個 Master Advisor 帳戶
2. 每位朋友開自己的 IBKR 個人帳戶（Sub-Account）
3. 朋友授權你為 Trading Authorization（有限交易授權）
4. 你從 Master 帳戶統一管理、統一下單
5. 系統自動按比例分配到各子帳戶
6. **朋友保有資金所有權**——錢在他們自己帳戶，你只有交易權限，不能轉帳提款

### Trading Authorization vs. Power of Attorney

| | Trading Authorization（推薦） | Power of Attorney |
|---|---|---|
| 範圍 | 只能交易 | 可以交易 + 轉帳 + 提款 |
| 信任需求 | 中 | 高 |
| 朋友控制 | 朋友隨時可撤銷 | 朋友隨時可撤銷 |
| 適合情境 | 代操帳戶 | 完全託管 |
| **建議** | ✅ 用這個 | ❌ 太大權限，容易有糾紛 |

### 台股問題

IBKR 不支援台股交易。解法：
- **方案 A**：朋友在台灣券商（元大、凱基等）另開帳戶，給你「受任人」交易授權
- **方案 B**：只做美/港/日三個市場，台股朋友自己操作你給建議
- **方案 C**：用複委託帳戶（台灣券商的海外交易帳戶），但手續費較高

### 何時該考慮成立公司？

| 規模 | 建議 |
|------|------|
| AUM < $500K、< 5 人 | ❌ 不需要。Friends & Family 足夠 |
| AUM $500K - $2M、5-10 人 | 🟡 開始考慮。可以先用合約規範 |
| AUM > $2M 或 > 15 人 | ✅ 應該成立。IBKR 限制會卡住 |
| AUM > $5M | ✅ 必須成立。需要法律保護 + 稅務規劃 |

## 二、收費模式設計

### 你的原始想法 vs. 我的建議

| 項目 | 你的想法 | 業界標準 | 我的建議 |
|------|---------|---------|---------|
| 管理費（月繳） | 1.5%/月 | — | ❌ 太高（年化 18%） |
| 管理費（年繳） | 1.2%/年 | 1.0-1.5%/年 | ✅ 合理 |
| 績效費門檻 | 8-10% | 5-8%（hurdle rate） | ✅ 8% 合理 |
| 績效費分成 | 超額 80:20 | 80:20 是業界標準 | ✅ 完全合理 |

### ⚠️ 重要修正

**你的月繳 1.5% 是月費率還是年費率？**

- 如果是 **1.5%/月** = **年化 18%**，這太高了。對方一年要付 18% 管理費，你的績效要超過 18% 朋友才開始賺錢。
- 如果是 **1.5%/年分月收** = 每月 0.125%，這是合理的。

### 推薦收費方案

```
方案 A：簡單版（適合朋友 2-3 人、AUM < $500K）
───────────────────────────────────────────
管理費：1.2% / 年（月繳 = 每月 0.1%）
績效費：超過 8% 年化報酬後，超額部分的 20%
高水位線：有（虧損年不收績效費，需先補回虧損才重新計算）
結算週期：每季結算一次

範例：
  朋友投入 $100,000
  年底帳戶值 $120,000（+20%）
  管理費：$100,000 × 1.2% = $1,200
  績效費：超額 = 20% - 8% = 12%
          12% × $100,000 = $12,000
          你收 20% = $2,400
  你的總收入：$1,200 + $2,400 = $3,600（佔對方獲利的 18%）
  朋友實際獲利：$20,000 - $3,600 = $16,400（+16.4%）
```

```
方案 B：進階版（適合 AUM > $500K）
───────────────────────────────────────────
管理費：
  - AUM < $200K：1.5% / 年
  - AUM $200K-$500K：1.2% / 年
  - AUM > $500K：1.0% / 年
績效費：超過 6% hurdle rate 後，超額的 20%
高水位線：有
結算週期：每半年
鎖定期：最低投資 6 個月

範例：
  朋友投入 $300,000
  年底帳戶值 $360,000（+20%）
  管理費：$300,000 × 1.2% = $3,600
  績效費：超額 = 20% - 6% = 14%
          14% × $300,000 = $42,000
          你收 20% = $8,400
  你的總收入：$3,600 + $8,400 = $12,000（佔對方獲利的 20%）
  朋友實際獲利：$60,000 - $12,000 = $48,000（+16%）
```

### 高水位線（High Water Mark）說明

```
假設時間軸：
  Q1：$100K → $110K（+10%）→ 收績效費 ✅
  Q2：$110K → $95K（-13.6%）→ 不收績效費 ❌
  Q3：$95K → $105K（+10.5%）→ 不收績效費 ❌（還沒回到 $110K）
  Q4：$105K → $115K（+9.5%）→ 只對 $110K → $115K 的部分收績效費 ✅

高水位 = $110K → $115K（持續更新）
```

**這個機制保護朋友：你不會在虧損後的反彈中重複收費。**

### 你需要準備的合約要素

- [ ] 管理費率和收取方式
- [ ] 績效費率、hurdle rate、高水位線規則
- [ ] 結算週期（每季/每半年）
- [ ] 最低投資金額和鎖定期
- [ ] 終止條款（提前退出的通知期，建議 30 天）
- [ ] 免責聲明（投資有風險，過去績效不保證未來）
- [ ] 報告義務（月報/季報）

## 三、投資管理流程（參考私募基金）

### 每日流程

```
08:00 ┌─ 市場掃描（15 分鐘）
      │  ├─ python3 run_tsi.py          → 科技壓力
      │  ├─ python3 run_dashboard.py    → CPI + AVI
      │  └─ 檢查隔夜市場（美股收盤、亞洲開盤）
      │
08:15 ├─ 風險檢查（5 分鐘）
      │  ├─ TSI > 50？→ 檢查科技持倉
      │  ├─ CPI > 35？→ 檢查整體風險
      │  └─ 任何 Flash Alert？→ 立刻行動
      │
08:20 ├─ 交易決策
      │  ├─ 是否需要調整倉位？
      │  ├─ 新的買入/賣出機會？
      │  └─ 避險工具是否需要調整？
      │
09:00 ├─ 執行交易（開盤時段）
      │  └─ 透過 IBKR Master Account 統一下單
      │
21:30 └─ 美股開盤監控（如需要）
         └─ 設定止損/止盈訂單
```

### 每週流程

```
週一：
  - 本週經濟數據/財報日曆檢查
  - 持倉 review（各帳戶）
  - 風險曝露計算

週五：
  - 週報撰寫（給朋友看的）
  - 下週策略調整
  - TSI/CPI 週回顧
```

### 每月流程

```
月初：
  1. 計算上月各帳戶績效
  2. 計算管理費（自動從帳戶扣除或另外收取）
  3. 生成月報（包含：報酬率、持倉明細、風險指標、市場展望）
  4. 跑 AVI V5 月度更新

月報模板（給朋友）：
  ┌─────────────────────────────────────────┐
  │  [名字] 投資月報 — 2026 年 X 月         │
  ├─────────────────────────────────────────┤
  │  本月報酬率：+X.X%                       │
  │  累計報酬率：+XX.X%                      │
  │  本月操作：買入 NVDA, 賣出 AAPL...       │
  │                                         │
  │  風險指標：                              │
  │    AVI: X.X/10  CPI: XX/100  TSI: XX   │
  │                                         │
  │  下月展望：...                           │
  │  管理費本月：$XXX                        │
  └─────────────────────────────────────────┘
```

### 每季流程

```
季末：
  1. 績效結算
  2. 計算績效費（如有超過 hurdle rate）
  3. 季度報告（更詳細，含 benchmark 比較）
  4. 與朋友 review meeting（30 分鐘）
  5. 下季策略調整
```

### 風險管理規則

| 規則 | 具體做法 |
|------|---------|
| **單一持股上限** | 任何一檔不超過 portfolio 的 15% |
| **單一產業上限** | 不超過 40% |
| **止損規則** | 個股 -15% 觸發檢討，-20% 強制停損 |
| **整體回撤上限** | 季度 -10% → 降低曝險至 50% |
| **現金最低比例** | 任何時候至少 5% 現金 |
| **避險最低比例** | CPI > 35 時至少 3% 用於 put options |

### 投資決策流程

```
新投資想法
    │
    ▼
[基本面分析]
    ├─ 財務數據（營收、利潤、現金流）
    ├─ 估值（PE、PS、DCF）
    └─ 產業趨勢
    │
    ▼
[技術面確認]
    ├─ 趨勢方向（均線、支撐/阻力）
    └─ TSI/CPI 環境確認
    │
    ▼
[風險評估]
    ├─ 倉位大小（Kelly criterion 或固定 %）
    ├─ 止損設定
    └─ 與現有持倉的相關性
    │
    ▼
[執行]
    ├─ IBKR 下單
    ├─ 記錄交易日誌
    └─ 通知相關帳戶持有人（如大幅調整）
```
