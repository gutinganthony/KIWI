# Recovery Camp MVP 計畫（v0 草案，autoplan 審查中）

## 1. 構想摘要（Jake 原始描述）

現代人工作/生活壓力大，需要一個世外桃源短暫逃離、治癒身心靈。做一個 camp：好吃的食物、瑜伽、頌缽、身心靈課程、其他體驗課。目標客群：**都會白領精英**，型態：**週末 1–2 天**，價值主張：**「充滿電、滿血回去面對都市生活」**。

既有資源：
- 宜蘭：朋友的舒適空間可借用（成本近零的場地）
- 人：朋友會催眠與頌缽（內容供給）
- 通路起點：身邊有此需求的朋友（首批訪談對象與種子客群）

## 2. 明定前提（審查對象——每一條都可能是錯的）

| # | 前提 | 若為假的後果 |
|---|---|---|
| P1 | 都會白領存在「短期逃離充電」的未滿足需求，且**願意為此付費**（而非用既有替代品：溫泉旅、按摩、爬山、耍廢） | 整個生意不成立 |
| P2 | **週末 1–2 天過夜**是對的產品形態（vs 半日工作坊／平日晚間／一週長營／線上） | 產品形態重做 |
| P3 | **身心靈內容組合**（瑜伽＋頌缽＋催眠＋好食物）是需求的正確解（vs 純休息放空／自然體驗／社交型／運動型） | 內容重組，朋友資源價值下降 |
| P4 | **宜蘭**是對的區位（台北出發 ~1 小時，心理上「夠遠又夠近」） | 區位重選 |
| P5 | 朋友資源（空間＋催眠/頌缽師）的品質**足以支撐付費產品**，不只是朋友情面 | MVP 品質不過關，燒掉種子客群信任 |
| P6 | 訪談 8–12 位目標客群＋預售測試，足以在 pilot 前驗證需求與定價 | 驗證方法重設計 |

## 3. MVP 設計草案（待研究數據與訪談校準）

- **形態**：2 天 1 夜 pilot，8–12 人小團
- **內容骨架**：抵達放鬆 → 瑜伽 → 質感晚餐 → 頌缽夜療 → 晨間靜心/自然 → 催眠或工作坊 → 收尾整合 →回程（課程與留白比例待研究對照）
- **定價假設**：待台灣行情研究定錨（目前未定，訪談用 Van Westendorp 校準）
- **成本結構假設**：場地（朋友借用，pilot 期近零）＋餐食（外燴/私廚）＋講師分潤＋保險＋雜項
- **通路假設**：朋友圈與轉介紹起步 → IG／Accupass 類平台
- **驗證漏斗**：訪談（8–12 人）→ 問卷放大（30+ 人）→ **預售 pilot（真金白銀訂金為最終驗證）**

## 4. 交付物清單

1. 市場研究三件套：全球趨勢與需求／現行商業模式／台灣案例（研究 agent 執行中）
2. **顧客訪談訪綱**（Mom Test 原則：問過去行為不問假設意見、最後才揭露概念、以具體承諾為驗證訊號）＋簡版問卷
3. autoplan 審查產出（前提裁定、替代方案、風險註記、營運面檢查）

## 5. NOT in scope（初版）

- 購置/長租場地、裝修
- 品牌官網與正式公司設立（pilot 前不需要）
- 多據點/加盟/會員制設計
- 療癒師證照體系與醫療宣稱（明確迴避醫療效果宣稱——法規紅線）

## 6. Open Questions（審查與研究要回答的）

1. 台灣既有供給長什麼樣？價格帶？「非宗教、質感餐食、白領定位」的空白是否存在？
2. 週末 retreat 的盈虧平衡人數與定價下限？
3. 「催眠」在台灣市場的信任度與法規/形象風險？
4. 需求是「逃離」還是「內容」？（決定行銷語言與產品重心）
5. 競品的替代品（溫泉飯店、免費禪修）如何影響定價天花板？
