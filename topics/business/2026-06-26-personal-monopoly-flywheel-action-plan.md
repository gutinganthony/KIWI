## 0. 一句話策略

> **不要再把自己想成「在做某個產品（AVI / SaaS / app）」；你就是產品。** AVI/Serenity 是你的「專有引擎」，國泰履歷是「信用憑證」，中文敘事是「介面」。把這顆腦袋 + 這套框架，沿著「免費觸及 → 付費研究 → 管理資本」三層漏斗變現，**同一份研究，三層收錢**，B2B 授權則是不佔你時間的被動加碼。

**核心紀律（貫穿全文）**：你已把「建造」練到滿級，缺的是「向某個人收到第一筆錢」。本計畫所有動作的北極星只有一個 → **付費人數**。不是再多一次回測，不是再多一版引擎。

## 1. 飛輪：一次研究，三層變現

```
                      ┌─────────────────────────────────────────┐
                      │   專有引擎：AVI · CRI · TSI · Serenity    │
                      │   信用憑證：國泰私銀 · JPMAM · 募資40億   │
                      └───────────────────┬─────────────────────┘
                                          │  (一次研究產出)
        ┌─────────────────────────────────┼─────────────────────────────────┐
        ▼                                 ▼                                 ▼
   ┌─────────┐   導流轉付費   ┌─────────┐   篩選 anchor LP   ┌─────────┐
   │   L1    │ ───────────▶ │   L2    │ ─────────────────▶ │   L3    │
   │ 免費觸及 │              │ 付費研究 │                    │ 管理資本 │
   │ 漏斗頂端 │ ◀─喊對紀錄回灌─│ 現金流   │ ◀──基金績效當內容──│ carry   │
   └─────────┘              └─────────┘                    └─────────┘
        │                        │                              │
        └──────────── B2B 授權層（媒體 / 券商 / 機構 / 課程）─────┘
              （把同一套引擎與內容，賣給「機構」而非「個人」）
```

**為什麼這個結構會贏**：三層用的是**同一套引擎、同一顆腦袋**。L1 內容餵養 L2 訂閱；L2 的受眾與信任是 L3 的天然 LP 池；L3 的真實績效又回頭成為 L1/L2 最強的內容。邊際成本趨近於零，信用卻會複利。**這就是「個人壟斷」**——全台同時擁有「機構信用 + 自建量化系統 + 內容分發 + 領域縱深 + 坐在 UHNW 錢旁邊」這五項的人，不到兩位數。

**台灣已有活證據**：財經 M 平方（MacroMicro）證明了「中文、方法論驅動、付費的總經/市場數據訂閱」在台灣是真生意、有上萬付費會員。你的差異化＝**他們偏總經數據儀表，你偏「可行動的風險分數 + 個股 alpha + 真人信用 + 未來真錢績效」**。市場被驗證過，你切的是更窄、更深、單客單價更高的一刀。

- Slogan（擇一）：**「看得懂的機構級風險情報。」** / *Institutional-grade risk intelligence, made legible.*

### 2.2 品牌（待你拍板，見 §11）

### 2.3 法規紅線（go/no-go，務必內化）

## 4. L2 — 訂閱層｜現金流 + 驗證引擎（最快見血）

**目的**：**經常性收入 + 你唯一缺的「真實付費訊號」+ L3 的 LP 線索池**。這是整條飛輪的鎖心石。

### 4.1 受眾修正（很重要）
**別賣給窮創作者**（你自己在 Copilot 文件已承認他們付費意願極低）。**把研究直接賣給會付錢的人**：
- 進階 prosumer 散戶（每月主動管理部位、自己看總經）
- 小型家辦研究員、獨立投顧、財管顧問（拿你的研究當底料）
- **與你 L3 的 LP 池高度重疊的高資產個人** ← 這層的隱藏價值：**L2 訂閱戶名單，就是你未來基金的暖名單**。

### 4.2 產品與定價（階梯）

| 方案 | 內容 | 定價（建議） | 對象 |
|---|---|---|---|
| **入門** | 每週風險判讀全文 + 每日讀數 + Flash Alert | NT$300–500/月 或 NT$3,000–5,000/年 | 一般進階散戶 |
| **進階** | 入門 + **Serenity 選股清單**（完整名單與評分）+ 模型更新 + 月度線上 Q&A | NT$10,000–30,000/年 | 認真投資人、獨立投顧 |
| **高階/小圈** | 進階 + 私密社群（LINE/Discord）+ 季度 1:1 或小組深談 | NT$50,000–100,000/年 | 高資產個人、準 LP |
| **機構** | 團體席次 + 客製 → 併入 §6 B2B | NT$20萬–100萬/年 | 投顧/家辦/券商研究部 |

> 算術：**300 個入門+進階混合、平均 NT$1.5 萬/年 = NT$450 萬/年**，已超過你國泰全包（~NT$250 萬）。而這 300 人裡只要 3–5 個轉成 anchor LP，L3 就點火。

### 4.3 平台與金流
- 最省力起步：**方格子付費訂閱** 或 **Substack paid**（內建會員 + 金流 + email，零開發）。
- 進階自主：自架（簡單站）+ **綠界/藍新/TapPay**（台灣）或 **Stripe**（跨境/收外幣）。
- 社群層：**LINE 社群 / Discord**（高階方案的留存與網絡效應引擎，也是 LP 觀察池）。

### 4.4 L2 的 90 天（北極星：付費人數）
- [ ] **法規前置**：先和投顧法律師確認 L2 收費的措辭/主體（見 §2.3）。在確認前，可先用「**創始會員候補名單 + 預購意向**」收集付費訊號而不實際收款。
- [ ] **30 天**：照你既有訪綱跑 **10–15 場訪談**（對象換成「會付錢的進階投資人/小家辦」，非窮創作者）。目標：5 人付費訊號 ≥2、3 人主動問「怎麼買」。
- [ ] **60 天**：開 **創始會員**（限量 + 早鳥價）。目標 **第一個付費用戶**。MVP 可以只是 email + Notion，別過度建造。

## 6. B2B — 授權層｜不佔你時間的被動/高單價加碼（你特別要求展開）

**核心心法**：L2 是「把研究賣給**個人**」，B2B 是「把同一套引擎與內容賣給**機構**」——單客單價 10–100x，且多半是 B2B2C（機構幫你分發）。**但 B2B 的信用前提，多半要 L1/L2 先建立。** 故下表標了優先序與「現在 vs 之後」。

| # | 機會 | 客戶（具體） | 賣什麼 | 商業模式 | 你的投入 | 優先序 |
|---|------|------|--------|----------|----------|:---:|
| B1 | **演講 / 企業內訓 / 工作坊** | 券商研究部、投信投顧、家辦、商學院 EMBA、財經社群 | 「AI × 量化風險研究」方法論、市場風險判讀 | 場次費 NT$3萬–20萬/場 | 低（講你已會的） | **現在** ⭐ |
| B2 | **TradingView 付費/邀請制指標** | 全球散戶 | AVI/WaveTrend 指標（**你已有 pine export**） | 訂閱分潤 + 導流 L1/L2 | 低（素材已在） | **現在** ⭐ |
| B3 | **線上課程（一次性高單價）** | 進階散戶、想學量化的人 | 把方法論做成課（風險判讀 / Serenity 框架） | 一次性 NT$3,000–15,000 + 名單 | 中（錄一次賣多次） | **現在/60天** ⭐ |
| B4 | **白標「市場風險溫度計」widget** | 財經媒體（鉅亨/MoneyDJ/股感/M平方型）、券商 app、理財平台 | 嵌入式 AVI/CRI 即時元件 + 中文敘事 | 年授權費 NT$30萬–200萬/家 | 中（需小整合） | 之後（需信用） |
| B5 | **資料 API / 研究底料授權** | robo-advisor、理財 app（富果/CMoney/三竹型）、內容平台 | AVI/CRI/TSI/Serenity 分數 feed | API 年訂閱（按席次/呼叫） | 中（需 API 化） | 之後 |
| B6 | **機構團體訂閱** | 投顧、家辦、券商研究部、私銀團隊 | L2 進階版的團體授權 + 客製簡報 | NT$20萬–100萬/年/機構 | 低（L2 內容複用） | 之後（L2 之後） |
| B7 | **白標研究 / 旗艦報告代筆** | 券商、投顧、家辦的品牌研究 | 用你的引擎產出他們掛名的報告 | 專案費 / 月 retainer | 中–高 | 機會財 |
| B8 | **策略出場 / 併購** | 財經媒體、券商、M平方型平台 | 收購引擎 + 品牌 + 名單 | 一次性對價 / earnout | — | 長期選項 |

**B2B 的取捨**：
- **現在就做、ROI 最高**：B1（演講/內訓）、B2（TradingView 指標）、B3（課程）。三者**零到低開發、立刻變現、同時替 L1/L2 引流並累積信用**。B1 尤其關鍵——一場對券商研究部的演講，既是收入、也是 B4/B6 的銷售前奏。
- **L1/L2 建立信用後再推**：B4（widget）、B5（API）、B6（機構訂閱）——這些客戶要先「看到你被市場認可」才會簽。
- **機會財 / 別主動追**：B7（代筆，小心稀釋個人品牌與利益迴避）、B8（併購，是結果不是目標）。

**法規提醒**：B 系列若涉及「對機構提供分析」相對乾淨（非對不特定公眾、非個股進出建議），但**收款主體**仍要避開國泰兼職條款（見 §2.3）——初期偏向「一次性演講費/課程」這類最不像「持續投顧行為」的形態。
