# 摸魚記 寫作風格指南

Jake Chen 的文章語氣與寫法，供 AI 協作時參考。

---

## 開頭結構：永遠從客戶對話切入

格式固定：
1. 時間點 + 客戶傳訊息 or 問我（具體問題，不要泛泛）
2. 我沒有直接回答他，改做別的動作（給他看圖、給他看數字、沉默等）
3. 他的反應（通常是一個問題或一個判斷）
4. 我的回應：**先承認他說的有道理，再轉折**

✅ 範例語氣：
> 「美光要破千了，是不是應該要加碼？」  
> 我沒有直接回答他。我給他看了一張圖。  
> 他看了大概十秒，說：「所以現在這黃燈是危險嗎？」  
> **對也不對。** 黃燈不是紅燈。但黃燈的意思是：可能還不用跑，但要知道你在幹嘛。

**關鍵細節：**
- 客戶問的問題要具體（股票名稱、具體動作），不要說「要不要投資」這種廢話
- 結論句要短、要有轉折、要帶一點生活感（「知道你在幹嘛」比「了解自己的位置」更好）

## 數字要精確，不要圓整

❌ 不要：「S&P 500 漲了很多」、「大約七千五百點」  
✅ 要：「7,599.96 點」、「-64,673 口」、「歷史 97th 百分位」、「5,658 億」

數字的精確感是建立信任的方式，讀者看到精確數字才覺得你是真的在看盤，不是在講故事。

---

## 市場新聞段落：三個點，每個一行，每個都有後果

格式：**粗體標題 + 一到三句說明 + 後果或張力**

不要只陳述事實，要說清楚「這件事為什麼重要、和另一件事有什麼關聯」。

✅ 範例結構：
> **荷姆茲海峽仍然半封鎖。** ……每天大概 2000 萬桶……油價站上每桶 100 美元……
> **新任聯準會主席上任了。** ……PCE 升到 3.8%……升息的機率是 70%。升息，不是降息。

最後收尾用一句話點出整體張力，不給結論：
> 「這三件事放在一起，和『S&P 創新高』不是矛盾——但確實是某種張力。」

---

## 指標說明：數字先，解釋後，不要說教

格式：先列數字 → 再說這代表什麼 → 再說歷史上這個情境通常怎樣

✅ 範例：
> AVI 5.8、CRI 17、TSI 45。  
> 這組數字和 2000 年的「三指標全紅」明顯不同……但 CRI 是 17，信貸市場沒有同步的壓力訊號。  
> 這就是為什麼我說黃燈，不是紅燈。

不要解釋太多技術細節，點到為止。讀者不需要知道 CRI 怎麼算，只需要知道 17 代表什麼感受。

## 慣用語彙（保持一致）

| 說法 | 意思 |
|------|------|
| 氣味 | 市場訊號的感覺，非量化的直覺 |
| 引線 | 觸發崩盤的事件 |
| 發散 | 兩個指標 / 兩種行為走反方向 |
| 容錯空間很薄 | 市場沒有辦法承受任何意外 |
| 在轉 | 某個引擎 / 動能還在運作 |
| 偏空 / 偏多 | 期貨方向性 |
| 黃燈 / 紅燈 | 風險程度，是全文貫穿的比喻 |

## 私銀茶水間段落：客戶行為就是市場信號

這段不分析數字，分析**人的行為模式**。

寫法：
1. 說客戶問我的問題「從 X 變成 Y」
2. 或說客戶行為從 A 模式轉到 B 模式
3. 點出這個轉變本身就是信號
4. 不要給結論，給觀察

✅ 範例：
> 最近客戶問我的問題普遍有兩種，一是從「有什麼機會」變成了「我的錢安全嗎」；二是從本來用大型科技股變成貪婪地用更波動的中小型股票去承作結構型商品。  
> 這個問題的轉變本身，就是一種信號。  
> 不是虛恐慌，但得謹慎。……恐慌幾乎都從謹慎與貪婪的分化開始。

## 結尾框架：給問題，不給答案

不要說「你應該怎麼做」，而是說「你應該問自己什麼問題」。

✅ 標準結尾邏輯：
1. 「沒辦法給具體建議，因為每個人不一樣」
2. 給一個思考框架（通常是一個反問）
3. 歷史對照（XXXX 年的人也是這樣說的）
4. 現在的定位：「我們還在……的過渡期，只是越來越靠近出口」

最後一句不要太聳動，要帶點開放感。

---

## 免責聲明的寫法

放在最後，非常簡短，用口語，不要法律語言：
> 「本文僅為個人觀點不構成投資建議。」

不要加很多字，不要說「過去績效不代表未來」這種廢話。

---

## 整體語氣原則

- **有觀點，不販賣恐懼**：說黃燈，解釋為什麼不是紅燈
- **有數字，不賣弄**：數字是佐證，不是展示
- **有距離感，不冷漠**：你是在旁邊陪讀者想清楚，不是在上課
- **口語中帶專業**：「知道你在幹嘛」「不是小數目」「腐爛」這些詞是刻意的
- **不誇大，不掩飾**：信號就說信號，不確定就說不確定
- **結論留給讀者**：你提供框架和數據，讀者自己下判斷
