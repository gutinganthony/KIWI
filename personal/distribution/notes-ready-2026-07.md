### Note 2（快層 vs 慢層）
```
VIXTWN 站上 44，比 6/11 那次危機底的 43.58 還高。

那為什麼我還是說「先試單，別重押」？

因為恐慌分兩層。

快層是法人、選擇權、外資：VIXTWN 44、外資連四天賣超破 1,700 億，這層已經投降了。

慢層是散戶：融資維持率還有 174.57%，離斷頭區的 130% 遠得很。

散戶根本還沒痛。

一個只有快層投降、慢層氣定神閒的底，是外部利空砸出來的，不是真正的籌碼洗乾淨。
```

### Note 3（V 型還是 L 型刀）
```
接刀之前，先問一個比「幾點可以買」更重要的問題：

這把刀，是 V 型的，還是 L 型的？

關稅、地緣、純情緒殺出來的，是 V 型：跌得快、回得也快，可以動作積極一點。

對 AI 到底賺不賺錢、結構出問題的懷疑，是 U 型或 L 型：慢慢跌、慢慢磨，要分更多批、留更多子彈。

你現在手上的倉位，是按哪一種刀在配置的？

還是其實你沒想過這個問題？
```

### Note 5（台積電除息是內資承接力的試金石）
```
台積電除息這週最該看的，不是它漲多少跌多少。

是它能不能「填息」。

填息，代表內資（ETF 大軍加散戶）的承接力還在，外資賣壓被吃得下去。

貼息而且擴大，代表連最強的那道承接牆都軟了。

在外資連四天賣超破 1,700 億的當下，這一場除息，其實是台股慢層那盞燈的一次體檢。

結果我會在這週的摸魚記追蹤。
```

## 二、來源 W3（VIX>40 崩盤抄底框架 · 6/12）

### Note 6（VIX 從來不是底）
```
每次崩盤，都有人問：現在能買了嗎？

我習慣先反問一句：你查過，過去的底部，VIX 是多少嗎？

2018 年 12 月底：36.07。
2020 年 COVID 底：57.08。
2022 年升息熊市底：34.53。

歷史上每一次確認底部，VIX 至少 30，真正的恐慌底要 34 以上。

VIX 二十出頭的時候，市場只是「嚇到了」。

還沒到「投降了」。
```

### Note 7（費半崩盤隔年）
```
費城半導體指數三次大崩盤，隔年報酬：

2002 科技泡沫：當年 -47.85%，隔年 +80.95%
2008 金融海嘯：當年 -51.69%，隔年 +74.95%
2022 升息循環：當年 -35.09%，隔年 +67.13%

三次崩、三次隔年大漲。

看起來抄底穩賺？

問題是：2022 年跌到底之前，先要撐過九個月的持續下跌。

沒有人買在最低點。撐得住的人，才有隔年那個數字。

你撐得住嗎？
```

### Note 8（唯一通過所有檢驗的訊號）
```
一份 1951 到 2026 年、138 種指標組合的回測，用了統計上最嚴格的多重比較校正。

結論很反直覺：

通過全部檢驗的，只有一種組合。

「S&P 回檔達 10%、而且仍站在 200 日均線上方，持有 6 個月」。

勝率 95%，樣本 37 次，含 2020 和 2022 的樣本外 10 次全中。

關鍵是那條 200 日均線。它在幫你過濾「跌破了就回不來」的結構性空頭。

均線之上的恐慌，歷史上全部都回來了。
```

### Note 9（等待不是失去機會 · 長青）
```
崩盤的時候最難的，不是判斷，是忍住。

2018 年那次，第一次大跌之後，還要再震盪兩三個月才真正見底。

耐住的人，才等到後面 18 個月的翻倍。

我一直提醒自己一句話：

等待不是失去機會。等待是讓機會的代價更低。

看，常常比動，更難、也更重要。
```
