## 二、市場空缺分析

### 台灣中文財經內容光譜

```
散戶導向                                          機構導向
────────────────────────────────────────────────────────
小樂講股  美股大叔  Kenji              [空白]    外資報告
（個股）  （產業）  （基本面）                   （英文/付費）
                                         ↑
                                     摸魚記的位置
```

### 空缺的名字
**「有產業實務背景、寫給高收入專業人士的中文財經視角」**

這個空白存在的原因：
- 在私銀做的人沒有動機寫（主業收入夠、怕合規問題）
- 有財經知識的人大多沒有私銀視角
- 敢寫的人沒有這個背景，有這個背景的人沒有動機寫

### 目標讀者的真實痛點
年收 200-500 萬的醫生/工程師/律師/科技業：
- 有閒錢，但沒有管道接觸私銀等級的思維框架
- 看散戶分析覺得太淺，看外資報告太難/太貴/英文
- 不需要有人告訴他們買什麼股票，需要有人幫他們「看懂格局」
- 想知道「有錢人在想什麼」但進不了那個圈子

## 三、品牌切入點

### 核心定位
**「你在私銀茶水間才聽得到的那個視角」**

不是分析師，不是媒體，是那個在私人銀行家族辦公室工作、
每天跟台灣真正有錢的人談資產配置的人，
把他平常不會公開說的事，寫成一封信寄給你。

### 為什麼這個定位有壁壘
| 要素 | 市場上有的人 | Jake 同時具備 |
|------|------------|--------------|
| 私銀 IFA 從業 | 很多，但不寫 | ✓ |
| 量化風控系統（AVI/CPI/TSI） | 幾乎沒有 | ✓ |
| 7年海外經歷（荷/港/滬） | 部分 | ✓ |
| 創業視角（Yuni/Dashboard） | 少 | ✓ |
| 願意寫、能寫 | 少 | ✓ |

### 套用到摸魚記的語氣
- 不說「根據數據顯示」，說「我這週跟客戶聊到一件事」
- 不給結論，給框架：「有錢人不是這樣想的，他們是這樣想的」
- 不預測市場，說信號：「我的 AVI 現在告訴我這個，代表什麼」
- 茶水間口吻：你在跟一個朋友說話，不是在做簡報

## 四、發文策略

### 頻率：週刊（每週四晚上 9 點）
- 週刊建立閱讀習慣，競品幾乎全是週刊
- 篇幅控制在 1,000-1,500 字，不追求完美，追求一致
- 雙週 = 26篇/年，週刊 = 52篇/年，迭代速度差一倍

### 每期固定結構
```
摸魚三分鐘（100字）      ← 茶水間口吻開場，永遠是一個故事或觀察
─────────────────────
本期主題（1,000-1,200字） ← 四柱輪流
─────────────────────
私銀茶水間（100字）      ← 一個 HNW 觀察，不說客戶名字
─────────────────────
三個數字（50字，月度期） ← AVI / CPI 快照，每月一次
─────────────────────
下期摸魚（30字）         ← 懸念預告
```

### 四柱輪播節奏
每四週一循環：
1. 市場視角（AVI/CPI 系統）
2. 私銀視角（HNW 觀察）
3. 量化/國際框架（素材消化 + Jake 詮釋）
4. 創業誠實日記 or 驗證回顧

## 六、「破局文」策略

### 核心原則：先發信號，再等驗證

**不要等崩盤才馬後炮，要在崩盤前寫「我在觀察這些信號」。**

這比 Rubinstein 的 SVB 模式更有價值：
- 他是在崩盤當下寫 → 及時但被動
- 你是提前寫信號 → 主動、有預測力、更能建立信任

### 具體格式

**第一篇（主動出擊）：**
「我的系統正在發出警告：三個不能忽視的崩盤前信號」
- AVI 數值 + 歷史對比
- CPI 崩盤概率指數現況
- 私銀觀察到的 HNW 行為變化（資金流向、問題類型改變）
- 結尾：「我不知道什麼時候會發生，但我知道這些信號出現過幾次」

**後續 Update 文（視情況）：**
- 情況1：信號持續升高 → 「一個月後，信號現在在這裡」
- 情況2：修正發生 → 「我們說的發生了，但這才是真正重要的事」
- 情況3：信號消退 → 「信號回落了，這代表什麼」

### 為什麼這個策略更強

| 方式 | 時機 | 公信力 | 流量 |
|------|------|--------|------|
| 馬後炮 | 崩盤後 | 低 | 一次性 |
| 當下報導（SVB模式） | 崩盤中 | 中 | 高但靠運氣 |
| **提前信號（摸魚記模式）** | **崩盤前** | **高** | **持續累積** |

這個格式也可以變成固定欄目——每季一篇「信號追蹤」，
讓 AVI/CPI 系統成為摸魚記獨家的品牌資產。

## 七、收益路徑

| 訂閱數 | 動作 | 對應其他機會 |
|--------|------|------------|
| 0 | 建立一致性，每週出刊 | — |
| 50 | 開接諮詢（NT$3,000-5,000/小時） | 主業品牌延伸 |
| 100 | Dashboard beta 名單開放 | IFA SaaS 驗證 |
| 200 | 開付費訂閱 tier | 被動收入開始 |
| 300 | Deal Flow 雙邊名單自然形成 | 投資人+創業者媒合 |
| 500 | 第一個線上 Q&A / workshop | 直接變現+口碑 |

**核心洞見：** 200人的高品質名單 > 2,000人的散戶名單。
摸魚記的訂閱者就是你所有其他事業機會的信任基礎。

## 八、素材來源策略

| 來源 | 角色 | 用法 |
|------|------|------|
| Kenji San | 主要素材 | 台灣追蹤少，加 Jake 私銀詮釋後有稀缺感 |
| 英文 Twitter（Readwise） | 差異化素材 | 讀者看不到的視角，最強差異化 |
| 美股大叔 | 市場溫度計 | 確認市場共識在哪，不直接作為素材 |
| Jake 的 AVI/CPI/TSI | 獨家內容 | 任何人都複製不了的部分 |
| 私銀日常觀察 | 獨家視角 | 「客戶最近在問的問題」類型 |

**素材處理原則：**
他們寫的 = 事實 + 分析
你加的 = 私銀視角 + HNW 觀察 + AVI 角度 + 台灣本土連結
