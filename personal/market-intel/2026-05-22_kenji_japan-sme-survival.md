# 日本350萬家中小企業的破局之路

> 來源：Kenji San Substack（2026-05-22）
> 標籤：#日本 #中小企業 #供應鏈 #總體經濟 #接班問題

## 關鍵數據

- 中小企業佔日本企業總數 **99.7%**，僱用全國 **70%** 勞動力，貢獻 GDP **50%**
- 2025 年員工薪資漲幅 **4.6%**（30年新高），但近四成企業是業績未改善下被迫加薪
- 設備投資動機：不是為了擴張，而是「被迫維持」
- 數位化有效：引進工具企業中 54.5% 業績正面、67.6% 效率提升

## 核心壓力：四頭夾擊

1. **成本**：原料、電費、人工三漲，報價卻不敢動（怕客戶跑）
2. **人才**：司機、建築、物流等第一線人手嚴重短缺
3. **接班**：老闆七旬以上，找不到繼承人，事業面臨消失
4. **日圓貶值**：進口成本大漲（對出口型有利，對內需型是災難）

## 九條出路（Kenji 整理）

| 路線 | 方向 |
|------|------|
| 留住人 | 薪資、高齡再就業、外籍勞工（現佔就業 3%） |
| 數位化 | 政府補助金支援，生產力中位數僅大企業一半，有巨大提升空間 |
| 股權融資 | 眾籌、天使投資，不背債又能擴張 |
| 接班規劃 | 贈與稅/遺產稅緩徵、M&A 交棒（非關門，是傳承） |
| 細分市場 | 打「隱形冠軍」路線，大企業嫌小的領域 |
| 漲價 | 主動與客戶協商，談了才有可能，不談一定沒有 |
| 出海 | 日圓弱勢 = 出口機會，JETRO 有一萬家出口支援計畫 |
| 法規紅利 | 善用商工會免費諮詢，多數經營者不知道補助存在 |
| 經營者自身 | 能制定並執行計畫、開放管理的老闆，績效明顯更好 |
